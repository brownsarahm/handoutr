About & Control

  • About

    This page is completely static, which makes the content you type private. There is no backend or connection to 3rd party sources that read the content, you can use blacklight by the independent journalists at The Markup to check and confirm.

    Usage

    You can work in this page or elsewhere, as you prefer. On each textarea there is a guide button where you can learn more about the syntax and a preview button to see the text formatted or diagram rendered. Get a pdf that is filled in or ready to write on (eg on paper or a tablet) by printing or export the markdown as a file or to your clipboard.

  • Save your work

    Export with the content entered into the boxes

  • Work Elsewhere

    Export prompts only, without textarea content or most hints
